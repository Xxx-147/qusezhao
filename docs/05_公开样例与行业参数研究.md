# 公开样例与行业参数研究

本项目不能只围绕少量本地样片调参，需要使用公开样例和成熟工具文档建立通用规则。

## 已采用的公开资料

1. RawTherapee Film Negative
   - 重点：负片转换不是简单 RGB 反相；不同胶片、老化和拍摄条件会导致 RGB 通道响应不同。
   - 可借鉴参数：参考指数、红通道比例、蓝通道比例、取中性点后再白平衡。
   - 来源：<https://rawpedia.rawtherapee.com/Film_Negative>

2. Negative Lab Pro 基础指南
   - 重点：相机翻拍负片时，建议先从胶片边框取白平衡；同一卷/同一胶片类型可以同步设置。
   - 可借鉴参数：Border Buffer、Pre-Saturation、Tone、Color、AutoWarm。
   - 来源：<https://www.negativelabpro.com/guide/basics/>

3. darktable negadoctor
   - 重点：负片转换需要分开处理 Dmin、扫描曝光偏置、黑白密度和高光/阴影，不应只靠一个自动色阶。
   - 可借鉴参数：film base、Dmin、exposure bias、paper black/white。
   - 来源：<https://darktable-org.github.io/dtdocs/en/module-reference/processing-modules/negadoctor/>

4. Wikimedia Commons: Negative Positive-Picture
   - 重点：同一图中包含橙色负片、去色罩负片、正片，可拆分为公开测试样例。
   - 许可：Public domain。
   - 来源：<https://commons.wikimedia.org/wiki/File:Negative_Positive-Picture.jpg>

5. Wikimedia Commons: Kodak Ultramax 400 Grain Scan
   - 重点：公开的 Kodak Ultramax 400 负片扫描，用于测试片基/颗粒和密度处理。
   - 许可：CC0。
   - 来源：<https://commons.wikimedia.org/wiki/File:KodakUltramax400GrainScan01.tif>

## 对当前算法的影响

当前新增 `smart_auto` 模式：

- 自动尝试多组转换预设。
- 避免纯白边框被误判成片基。
- 评分时惩罚蓝青偏色、暗部压死、高光裁切、亮度过低或过高。
- 批量处理时可对每张图单独选择更稳定的预设。

下一步计划：

- 加入每通道指数曲线，接近 RawTherapee 的 reference exponent / red ratio / blue ratio。
- 加入 AutoWarm，专门修正反相后常见的蓝青偏色。
- 加入同卷 profile：先从一卷中多张图估计共享片基，再对每张单独调曝光和纸黑/纸白。
