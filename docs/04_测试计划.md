# 测试计划

## 1. 测试目标

验证程序能稳定完成彩色负片自动去色罩、转正片、自动校色和批量输出。

## 2. 测试类型

### 2.1 单元测试

覆盖：

- 片基估计
- 边框像素提取
- 通道归一化
- 反相
- 自动色阶
- 灰世界白平衡
- 参数校验

### 2.2 合成样例测试

构造已知正片和已知橙色片基，生成负片，再让程序转换回来。

验证：

- 估计片基 RGB 是否接近设定值。
- 输出是否接近原正片。
- 极端颜色区域是否溢出。

### 2.3 真实样片测试

准备不同情况：

- 带胶片边框
- 无胶片边框
- 过曝负片
- 欠曝负片
- 老胶片褪色
- 高反差场景
- 人像肤色
- 天空和白墙等中性区域

### 2.4 批处理测试

验证：

- 多图批量处理。
- 不支持格式跳过。
- 单张失败不会终止全部任务。
- 输出命名正确。
- 诊断日志完整。

### 2.5 GUI 测试

验证：

- 打开图片。
- 参数调节。
- 转换预览。
- 保存输出。
- 批量队列。
- 后台任务不阻塞界面。

## 3. 测试指标

| 指标 | 目标 |
|---|---|
| 单张 JPEG 处理时间 | 5 秒内 |
| 批处理失败隔离 | 单张失败不影响其他文件 |
| 片基估计误差 | 合成样例 RGB 平均误差小于 10 |
| 输出可复现 | 同参数同输入输出一致 |
| GUI 稳定性 | 连续处理 50 张不崩溃 |

## 4. 测试数据准备

建议目录：

```text
test_data/
  synthetic/
  real_with_border/
  real_without_border/
  expired_film/
  overexposed/
  underexposed/
```

每张真实样片记录：

- 胶片品牌
- 胶片型号
- 扫描设备
- 是否有边框
- 是否过曝/欠曝
- 期望效果备注
