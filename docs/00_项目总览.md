# 胶片自动去色罩自动化程序 - 项目总览

**项目名称：** Film Mask Automation  
**项目类型：** 桌面端自动化图像处理工具  
**当前阶段：** 立项与方案设计  
**编写日期：** 2026-06-03  
**目标用户：** 胶片摄影用户、翻拍扫描用户、影像整理工作室、需要批量处理彩色负片的人  

## 一句话目标

做一个桌面自动化程序，自动识别彩色负片的橙色片基/色罩，完成负片转正片、自动校色、批量输出，并保留人工微调入口。

## 背景

彩色 C-41 负片普遍带有橙色片基。扫描或相机翻拍时，数字图像会同时记录画面内容和橙色片基。如果只做 RGB 反相，橙色会变成蓝青色，最终正片会严重偏色。因此程序必须先估计色罩，再进行通道归一化、反相、色阶拉伸和白平衡。

## 立项价值

1. 降低胶片数字化门槛：用户不需要 Photoshop、Lightroom 或复杂曲线知识。
2. 提高批量效率：一卷胶片可以统一处理，减少重复调参。
3. 保留可控性：自动处理失败时，用户可以手动指定片基色、边框取样、曝光、对比度和白平衡。
4. 可扩展：第一阶段做桌面工具，后续可扩展为插件、Web 工具或手机端辅助工具。

## 项目交付物

第一阶段交付：

- 桌面自动化程序 MVP
- 单张图片转换
- 文件夹批处理
- 自动片基色估计
- 自动负片转正片
- 基础色阶、白平衡、伽马、对比度调整
- 输出处理日志和诊断数据
- 项目说明文档、使用手册、测试记录

后续阶段交付：

- 图形界面
- 预览和参数滑杆
- 胶片边框/中性灰取样点
- 每卷胶片预设
- 16-bit TIFF 支持
- RAW 翻拍工作流
- 褪色修复和灰尘划痕辅助检测

## 成功标准

MVP 成功标准：

- 用户可以选择一张彩色负片扫描图并输出正片。
- 用户可以选择一个文件夹并批量输出所有图片。
- 对带胶片边框的图片，程序能稳定估计橙色片基。
- 对无边框图片，程序能给出可接受的自动估计结果，并允许手动覆盖。
- 输出图片不出现明显蓝青色整体偏色。
- 所有处理参数可追踪，便于复现和调试。

## 当前假设

- 输入图片先支持 JPEG、PNG、TIFF、BMP。
- 第一版不直接处理相机 RAW 文件。
- 第一版使用 Python 实现核心算法和自动化流程。
- 桌面 GUI 可在核心算法稳定后再做，避免界面先行导致核心处理不稳。
