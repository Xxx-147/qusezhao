# AI 模型训练计划

## 目标

在现有规则算法和手动调节能力之外，训练一个可以自动完成负片去色罩、转正片、基础校色的图像到图像模型，并集成到 CLI 与 GUI。

## 训练集和测试集定义

正确的监督学习数据结构是：

```text
训练集：
  input  = 橙色负片
  target = 去色罩/转正片结果

测试集：
  input  = 另一批橙色负片
  target = 对应去色罩/转正片结果
```

不能把“训练集=负片、测试集=正片”分开理解。测试集也必须有负片输入，否则模型没有输入；测试集也必须有目标正片，否则无法计算误差。

### BlueNeg

BlueNeg 是目前找到的最接近项目目标的公开胶片负片数据集。

当前只下载 20 对 8-bit 小样本：

```text
datasets/blueneg_subset/blueneg_manifest.csv
```

注意：

- BlueNeg 16-bit DNG 原始负片是真正的负片输入，适合构建“橙色负片 -> 去色罩/正片”训练集。
- BlueNeg 8-bit preview 是负片经过初步反相后的预览，更适合训练后处理/修复；它不能作为“橙色负片 -> 去色罩正片”的主训练集。
- 当前真正的负片训练/测试使用 `datasets/true_negative_pairs/`。
- 完整负片到正片模型后续需要 BlueNeg DNG 子集或更多真实扫描成对数据。
- BlueNeg 使用需要署名：`Copyrighted by Tien-Tsin Wong`。

BlueNeg 下载方式：

```powershell
python tools\download_blueneg_subset.py --mode raw-negative --count 4 --output-dir datasets\blueneg_raw_subset
```

这会下载：

```text
negative-16bit/*.dng        # 真正负片输入
pseudogt-8bit/*.pseudogt.png # 对应目标
```

然后把 DNG 渲染成训练用 PNG：

```powershell
.\.venv-ml\Scripts\python tools\render_blueneg_raw_negatives.py datasets\blueneg_raw_subset\blueneg_raw_negative_manifest.csv --output-dir datasets\blueneg_raw_rendered
```

注意：DNG 原始负片很大，不建议一次下载完整 290GB。先从 4-8 对开始，确认预处理和训练质量，再扩大到 50、300、1000。

## 数据规模策略

| 阶段 | 数据量 | 目标 |
|---|---:|---|
| 管线验证 | 4-20 对 | 确认训练、保存、推理、GUI 集成 |
| 初步模型 | 20-80 对 | 学习基础色彩和明暗映射，不追求泛化 |
| 可用模型 | 300+ 对 | 建立跨场景、跨胶片、跨曝光的稳定能力 |
| 正式模型 | 1000+ 对或更多 | 批量自动化接近商业工具起点 |

## 模型路线

第一版：

- 输入：RGB 负片或规则算法初步输出
- 输出：目标去色罩/正片
- 模型：轻量残差 U-Net
- 损失：L1 + 简化 SSIM

第二版：

- 加入感知损失
- 加入色彩直方图损失
- 加入多尺度训练
- 加入 roll-level profile：同一卷共享片基/色彩条件

第三版：

- 使用更大的 U-Net 或 diffusion/transformer 图像修复模型
- 支持局部色偏、褪色、蓝通道劣化修复
- 输出模型版本和质量指标

## 手动模式保留原则

AI 模型不能替代手动模式。GUI 必须长期保留：

- 片基色/色罩模式
- 手动 RGB 片基
- 黑点/白点
- 参考指数
- 红通道比例
- 蓝通道比例
- 曝光
- 亮度
- 对比度
- 伽马
- 饱和度
- 色温
- 色调
- RGB 通道增益
- 白平衡开关
- 锐化
- Profile 校准

优先级：

1. AI 模型：选择模型时使用。
2. 智能自动：未选择模型且开启智能自动时使用。
3. 手动参数：关闭 AI 和智能自动时完整生效。

## 参考工具

- RawTherapee Film Negative：参考指数、红比例、蓝比例、中性点采样。
- darktable negadoctor：film base/Dmin、Dmax、阴影和高光色偏、print exposure、paper black、paper grade。
- Negative Lab Pro：border buffer、pre-saturation、color models、tone/color 控件、批量同步。
- negRGB / 类似 RGB 工具：按通道调节和颜色偏移修正。
